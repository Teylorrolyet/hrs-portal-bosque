# PORTAL BOSQUE

Sistema web para control manual de horarios.

## Uso

1. Descargar archivos
2. Subir a GitHub
3. Activar GitHub Pages
4. Abrir desde navegador

## Importante

Los datos se guardan solo en el dispositivo actual usando LocalStorage.
No se sincronizan entre equipos.
